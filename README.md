## Testing React Heroes App

<https://jestjs.io/docs/tutorial-react>

### 🚀 Quick start

#### Instalar dependencias

```bash
yarn
```

#### Correr aplicación localmente

```bash
yarn dev
```

#### Correr Pruebas

```bash
yarn test
```

#### Docker Image

```bash
docker build -t heroes-app . --no-cache

docker container run -p 80:80 heroes-app 
```

<http://localhost:80>
